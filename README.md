# launchpad-window
i make launchpad on window! yeah using python but the packages its easy, just 1 requirement so i pack it on a bat

# how to use:
so you first get python which is easy

and run requirement.bat

then run run.bat

enjoy!

# info
requirement: pywin32

ram eating: about 14MB

version the dev use: python 3.14.6 (so it will be better i guess)

disk eating: 18.9 KB

  on disk: 24 KB
